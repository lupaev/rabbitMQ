### Сервис по распределению вагонов через RabbitMQ
### Функционал:
Получаемые вагоны разпределяются по очередям RabbitMQ, после получения уже 
распределенных вагонов  происходит сохранение в БД.

### Используемые технологии:
- Java 17
- Spring Boot 3.3.0
- Postgres 16
- Liquibase 4.27
- MupStruct 1.5.5
- Lombok 1.18.28
- Gradle 8.7
- RabbitMQ 3.13.3
